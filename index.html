Description

SixGrid a new simple launcher I made is out for public release. This launcher consists of a grid of six simple squares which each launch the most used android tasks which are your Applications, Browser, Messages, Camera, Dialer and even Settings! You can get the latest release (v1.0.1) at https://github.com/grkx/sixgrid/releases by getting the latest binary from the release. If you wish to contibute to this code then feel free to send me a pull request! Finally thank you for using SixGrid.
